장학제도
장학명 대상 및 선발기준 장학금액
신입생 재학생
성적 학년 최우수 장학금 (1학년 2학기 부터) 학년 성적이 가장 우수한 자 수업료 100% 감면
성적 최우수 장학금 학년 성적이 차순위로 우수한 자 수업료 70% 감면
성적 우수 장학금 학년 성적이 차차순위로 우수한 자 수업료 50% 감면
성적 장려 장학금 학년 성적이 차차차순위로 우수한 자 수업료 30% 감면
성적 준장려 장학금 학년 성적이 차차차차순위로 우수한 자 수업료 10% 감면
기술우수 장학금 기능경기대회 수상자로서 입학한 자
(신입생에 한함)		 전국대회 금상 : 수업료50%감면
은상 : 수업료 45%감면
동상 : 수업료 40%감면
지방대회 금상 : 수업료30%감면
은상 : 수업료 25%감면
동상 : 수업료 20%감면
공로 장학금 학생회 간부로 활동 중인 자 총학생회장 : 수업료 100%감면
부총학생회장 : 수업료 50%감면
총학집행부 : 수업료 25%감면
전공대표 : 수업료 20%감면
전공부대표 : 수업료 10%감면
전공 별 한년대표 : 수업료 10%감면
근로 장학금 근로학생으로 선발된 자 일반근로 : 700,000원
영상편집 : 500,000원
총장특별 장학금 대외적으로 학교의 명예를 높인 공로가 인정된 자
(전국규모 외부 공모전 등 수상 시)		 1등 : 300,000원
2등 : 200,000원
3등 : 100,000원
해외교류프로그램 지원 장학금 학교 내 해외교류 프로그램에 참가한 자
(해외봉사, 해외답사, 해외인턴십 등)		 미정
생활보호대상자 장학금 국민기초생활보호대상자로 심의 통과자 수업료 20%감면
형제자매 장학금 민법상 2촌 이내인 형제자매가 동시에 재학 중인 자 수업료 10%감면(형제자매 각각 10%감면)
보훈 장학금 국가/독립 유공자 중 재학 중인 자 수업료 100%감면
한성가족 장학금 한성여고를 졸업하고 입학한 자 입학 첫 학기 수업료의 20%감면
교직원자녀 장학금 교직원 자녀로서 성적이 우수하고 품행이 바른 자 수업료 100%감면
  • ※ 총장 특별장학금은 2017.03.01.부터 중복 수상 시 중복 수혜가능
    ※ 성적 학년최우수장학금의 경우 해당 학년이 50명이상일 경우에만 지급하며, 50명 미만일 경우에는 최우수 장학부터 지급함
    ※ 성적장학금은 2017-1학기부터 등록금에서 감면형태로 지급함
    ※ 성적장학금 외 모든 장학금은 등록금을 완납한 후에 지급됨
